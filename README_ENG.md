# 📊 Google Play Store Data Analysis

Exploratory Data Analysis (EDA) of 10,841 Android Apps from Google Play Store

> 📖 **Languages:** **English** | [Deutsch](README.md) | [Türkçe](README_TR.md)

## 📋 About the Project

Analysis of Android apps focusing on:

- App categories and ratings
- Pricing strategies (free and paid)
- User preferences and trends
- Temporal developments (2010-2018)

## 📦 Dataset

**Data Source:** [Google Play Store Apps - Kaggle](https://www.kaggle.com/datasets/lava18/google-play-store-apps)

**10,841 Apps** with 13 features: Name, Category, Rating, Reviews, Size, Installs, Type, Price, Content Rating, Genres, Last Updated, Current Version, Android Version

## 🔍 Key Findings

✅ Most popular categories: Family, Games, Tools  
✅ Best ratings: Entertainment, Education, Art  
✅ Strong growth between 2016-2018  
✅ Ratings concentrated at 4.0-4.5  
✅ Free apps dominate the market

Python • NumPy • Pandas • Matplotlib • Seaborn • Jupyter Notebook
